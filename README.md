# Overview

A template of Express.js Web Server.

## Technologies Used
Express.js: A minimal and flexible Node.js web application framework.

Node.js: Javascript run time environment.

## Getting Started

- How to start

## Install dependencies

```
npm install or npm i
```

## Start Server

```
npm run dev
```
